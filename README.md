## 🌟 About the Project

**Student Finance Tracker** is a friendly web app for Bangladeshi students to track:

- 💰 Income (allowance, tuition, freelance, etc.)
- 🧾 Expenses by category
- 📅 Monthly + date-range spending
- 📊 Visual category chart
- 🧠 Smart budget suggestions
- 🤖 AI buddy **Shokina**

Built to make money tracking simple, human, and student-life realistic.

---

## 🚀 Features

- ✅ Add / Edit / Delete **Income** and **Expense**
- ✅ Filter by **Month + Year**
- ✅ Date range filter (**start date / end date**)
- ✅ Category-wise spending and mini budget bars
- ✅ Smart warnings when category budget is near/exceeded
- ✅ AI helper (**Shokina**) with quick budget guidance
- ✅ Export current view to **CSV**
- ✅ Clean UI, mobile-friendly layout, single-file simplicity

## 🛠️ Run Locally

1. Download or clone this repo
2. Open `index.html` in browser  
   *(No build tools, no npm, no setup headache)*

---

## 🌍 Deploy on GitHub Pages

1. Go to **Repo → Settings → Pages**
2. Under **Build and deployment**
   - Source: `Deploy from a branch`
   - Branch: `main` (or your default) / root
3. Save
4. Your site will be live in a minute 🎉
